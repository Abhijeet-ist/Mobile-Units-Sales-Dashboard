# Mobile Sales Dashboard

A Power BI dashboard for tracking and analyzing mobile unit sales performance.

## What's Inside

- **Mobile Units Sales Dashboard.pbix.zip** — Power BI dashboard file with visualizations and metrics
- **Mobile Sales Data.xlsx** — Supporting sales data in Excel format

## Getting Started

### Requirements
- Microsoft Power BI Desktop (free version works)
- Excel (to view raw data)

### Setup

1. Extract `Mobile Units Sales Dashboard.pbix.zip`
2. Open the `.pbix` file in Power BI Desktop
3. The dashboard will load with data from the Excel file
4. If data doesn't load, check the data connection paths in Power BI settings

## Using the Dashboard

The dashboard displays:
- Sales volume and trends
- Unit performance metrics
- Key sales indicators

Refresh the data in Power BI when you update the Excel file.

## Data Updates

Edit `Mobile Sales Data.xlsx` to update sales figures. Refresh in Power BI to see changes.

## Questions or Issues?

Check the data source connections in Power BI if charts appear empty.
